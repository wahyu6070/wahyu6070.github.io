Litegapps is an application designed to optimize RAM usage on Android phones. This application can help users save RAM by separating system applications and third-party applications. So, users can choose which applications they want to install in it and which do not.

Project

Litegapps++ is an application developed by a third party to optimize RAM usage on Android phones. This application is a development of the previously well-known Litegapps, by adding new, more complete and quality features.

Litegapps Addon is an addon that works with the Litegapps Controller app, allowing users to install apps without adding an icon on the phone's home screen. This addon offers an alternative solution to reduce the application load on your phone without having to remove the application.
